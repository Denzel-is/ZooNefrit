# ZooNefrit

Адаптивный статический сайт интернет-магазина зоотоваров.

## Что внутри

- `index.html` — главная страница с hero-блоком, информацией о компании, акциями, поставщиками и отзывами.
- `catalog.html` — каталог со 100 товарами, поиском, фильтрами и модальным окном товара.
- `style.css` — стили сайта.
- `script.js` — массив товаров, фильтрация, карточки, модальное окно и кнопка «Наверх».
- `images/` — локальные SVG-изображения.

## Запуск

Откройте папку `zoo-shop-site` через Live Server и перейдите на `index.html`.
Сайт не использует backend, frameworks и внешние зависимости.
